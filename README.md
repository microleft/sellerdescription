## 为什么有这个脚本
淘宝因产品调整需要，从2019年4月18日起，卖家店铺档案页面将不再展示

[怎样看待淘宝将不再展示卖家店铺档案页面？](https://www.zhihu.com/question/319211686)

**其中有人写了个Chrome扩展。因为不想安装扩展，于是有个这个脚本。**
